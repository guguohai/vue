# vuescaffold

## Project setup
```
npm install
npm install -g @vue/cli
npm install -g @vue/cli-service-global
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
